# agentcarousel homebrew-tap

Homebrew tap for the [AgentCarousel](https://github.com/agentcarousel/agentcarousel) CLI

## Install

```bash
brew tap agentcarousel/tap
brew install agentcarousel
```

## Upgrade

```bash
brew update
brew upgrade agentcarousel
```

## Uninstall

```bash
brew uninstall agentcarousel
```
